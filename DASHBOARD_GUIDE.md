# Quick Start Guide - Streamlit Dashboard

## 🚀 Launching the Dashboard

1. **Install dependencies** (one time):
   ```powershell
   pip install -r requirements.txt
   ```

2. **Run the dashboard**:
   ```powershell
   streamlit run app.py
   ```

3. The dashboard will open automatically in your browser at `http://localhost:8501`

## 📁 Uploading Data

The dashboard accepts two types of CSV files:

### 1. Consumer Risk Scores CSV
Generated by the notebook (`outputs/consumer_risk_scores.csv`). Required columns:
- `consumer_id`: Unique consumer identifier
- `ensemble_score`: Risk score (0-1)
- `risk_category`: Critical, High, Medium, Low, or Minimal
- `ensemble_prediction`: Binary prediction (0 or 1)
- Optional: `autoencoder_score`, `lstm_score`, `rule_score`, `true_theft_label`

### 2. Raw Consumption Data CSV
Time-series electricity consumption data. Required columns:
- `consumer_id`: Unique consumer identifier
- `timestamp`: Date and time (e.g., "2024-01-01 00:00:00")
- `consumption_kwh`: Consumption in kWh
- Optional: `is_theft` (0 or 1 for labeled data)

## 🎨 Dashboard Features

### 📊 Overview Tab
- Total consumers and risk metrics
- Risk category distribution chart
- Ensemble score histogram

### 🎯 Risk Analysis Tab
- Top 15 high-risk consumers
- Risk category breakdown
- Model contribution scores (autoencoder, LSTM, rules)

### 📈 Model Performance Tab
- Confusion matrix
- Accuracy, Precision, Recall, F1-Score
- Available only if `true_theft_label` column exists

### 📋 Consumer Details Tab
- Sortable/filterable table of all consumers
- Filter by risk category
- Download filtered results as CSV

## 💡 Tips

- **Test with sample data**: Run the notebook to generate `outputs/consumer_risk_scores.csv`
- **Filter efficiently**: Use the risk category filter to focus on high-risk cases
- **Export results**: Download filtered consumer lists for investigation
- **Compare patterns**: Upload consumption data to see hourly patterns

## 🔧 Troubleshooting

**Dashboard won't start:**
```powershell
pip install --upgrade streamlit plotly
```

**"Unknown file format" error:**
Check that your CSV has the required columns listed above.

**Slow performance:**
Large CSV files (>100k rows) may take time to load. Consider filtering data before upload.
